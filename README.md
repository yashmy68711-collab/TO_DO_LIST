# Simple To-Do List (Python)

## About

This is a very basic command-line To-Do List application built using Python.
It allows users to add and view tasks.

## Features

* Add tasks
* Show all tasks
* Simple menu system

## Tech Used

* Python

## How to Run

1. Run the Python file
2. Choose an option from menu
3. Add or view tasks

## Future Improvements

* Delete tasks
* Save tasks in file
* Add GUI version
